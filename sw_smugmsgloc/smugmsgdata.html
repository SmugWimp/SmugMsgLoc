Message Data String
Message Data:
Enter a URL, or Message Default Data

By default the data is a Google URL;

http://maps.google.com/maps?q=[deviceLatitude],[deviceLongitude]

It is suggested that the URL remain. However, if you wish to change the format
use [deviceLatitude] and [deviceLongitude] as placeholders for the coordinates in your text.

For example: "Hey, I bet you can't find me! I'm at [deviceLatitude] and [deviceLongitude]. I'll give you an hour!"

Remember SMS Text Messages are typically limited to 160 characters.
Total Coordinate length 'up to' 21 (-XXX.XXXXX,-XXX.XXXXX) characters.
